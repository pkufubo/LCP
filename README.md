## 模块介绍

### 粮食农田模块

- **模块功能**

    预测某一年所需要的耕地面积。

- **基本约束**
  - 现行中国18亿亩耕地红线，全国划定永久基本农田15.5亿亩。
  
- **模块假设**
  - 人均对粮食、油料、棉花、糖类的需求不变
  - 我国种植各类农产品的耕地比例不变
  - 各类农产品的亩产逐年有稳定增长
  - 各类农产品的进口占比不变
